🔹 Setup & Config
git config --global user.name "Your Name"
git config --global user.email "you@example.com"
git config --list → check your config

🔹 Create / Clone Repository
git init → start a new repo in your folder
git clone <repo-url> → copy an existing repo from GitHub/GitLab

🔹 Basic Workflow
git status → see current changes
git add <file> → stage a file
git add . → stage all changes
git commit -m "message" → save changes locally

🔹 Branching
git branch → list branches
git branch <branch-name> → create new branch
git checkout <branch-name> → switch branch
git checkout -b <branch-name> → create + switch
git merge <branch-name> → merge into current branch

🔹 Remote Repositories
git remote add origin <repo-url> → connect repo
git remote -v → view remotes
git push origin <branch> → upload changes
git pull origin <branch> → download + merge
git fetch → get updates without merging

🔹 Undo / Fix Mistakes
git restore <file> → discard changes
git reset <file> → unstage file
git reset --hard → reset everything (⚠️ deletes changes)
git revert <commit> → undo a commit safely

🔹 Logs & History
git log → full history
git log --oneline → compact view
git diff → see changes

🔹 Stashing (Temporary Save)
git stash → save work temporarily
git stash pop → restore it

🔹 Tags
git tag → list tags
git tag v1.0 → create tag
